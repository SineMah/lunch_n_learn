# GraphQL Example

## installation & start
```
npm i
docker-compose up -d
npm run-script dev
```
You will need to create `_users` and `_replicator` for CouchDB.

### CouchDB admin UI

`http://127.0.0.1:5984/_utils`
```
User: admin
Password: secret
```

### GraphiQL UI 
`http://127.0.0.1:4242/graphql`
